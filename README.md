# MERN Auth System 🔐

A full-stack authentication system built using MERN stack with user registration, login, and logout functionality.

## 🚀 Features
- User Registration
- User Login
- User Logout
- Password Hashing using bcrypt
- JWT Authentication
- Protected Routes

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT, bcrypt
